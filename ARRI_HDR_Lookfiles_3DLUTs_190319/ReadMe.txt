// ARRI Look Files and 3D LUTs //


This directory contains a set of 3DLUTs and ALF-2 look files for HDR in the following folders

ARRI_Lookfiles_for_cameras

    ALF-2 look files for cameras. Please refer to the document "ARRI HDR.pdf"
    for further information.
  
  
3D_LUTs_for_cinema_grading

    Grading/mastering for DolbyVision cinema at 108 nits


3D_LUTs_for_monitor_grading

    Grading/mastering/preview on monitors with SMPTE 2084 (PQ) curve and
	BBC/NHK HLG curve according to ITU Rec. BT.2100. 
	HLG LUTs are supplied with 1000cd/m2 peak white.
	PQ LUTs are supplied with 1000, 2000, and 4000 cd/m2 peak white.


3D_LUTs_to_simulate_SDR_in_HDR

    3DLUTs and ALF-2 look files that simulate the standard rendering for 100 
    with PQ output. They can be used to switch between HDR and SDR image.
